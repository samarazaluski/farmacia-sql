# farmacia-sql
# 💊 Sistema de Banco de Dados para Farmácia

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos de **Banco de Dados** utilizando **MySQL**. A proposta é simular um sistema de uma farmácia, permitindo o cadastro de clientes, medicamentos e vendas, além da realização de consultas utilizadas no dia a dia de um Analista de Dados.

Este repositório faz parte do meu processo de estudos e da construção do meu portfólio para atuar como **Analista de Dados Júnior**.

---

## 🛠️ Tecnologias utilizadas

* SQL
* MySQL
* MySQL Workbench
* Git
* GitHub

---

## 📂 Estrutura do banco

O banco de dados é composto por três tabelas principais:

### 👥 Clientes

Armazena informações dos clientes da farmácia.

Campos:

* id
* nome
* idade
* cpf
* telefone

---

### 💊 Medicamentos

Armazena os medicamentos disponíveis para venda.

Campos:

* id
* nome
* fabricante
* preco

---

### 🛒 Vendas

Registra as vendas realizadas.

Campos:

* id
* cliente_id
* medicamento_id
* quantidade
* data_venda
* total
